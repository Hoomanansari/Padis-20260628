# ابهامات و پرسش‌های باز پروژه

این سند برای ثبت پرسش‌هایی است که باید از کارفرما یا تیم فنی دریافت شود.

## 1. اطلاعات کارفرما و پروژه
- نام رسمی کارفرما چیست؟
- نام رسمی مدرسه یا مجموعه آموزشی چیست؟
- کشور، شهر و آدرس پروژه چیست؟
- مخاطب اصلی پروپوزال چه شخص یا واحدی است؟

## 2. پلان و سایت
- آیا فایل AutoCAD پلان‌ها موجود است؟
- آیا پلان‌های دریافت‌شده آخرین نسخه هستند؟
- آیا کاربری همه اتاق‌ها در پلان‌ها نهایی و تأیید شده است؟
- آیا مسیرهای واقعی ورود و خروج دانش‌آموزان، والدین، پرسنل و سرویس مدرسه در پلان مشخص شده‌اند؟
- آیا بازدید میدانی انجام خواهد شد؟

## 3. کنترل دسترسی
- تعداد دقیق درب‌هایی که باید کنترل شوند چقدر است؟
- کدام درب‌ها نیاز به کنترل یک‌طرفه دارند و کدام دوطرفه؟
- برای هر درب، نوع قفل و وضعیت برق/شبکه مشخص است؟
- آیا همه کلاس‌ها به کنترل دسترسی الکترونیکی نیاز دارند یا برخی فقط مانیتورینگ می‌شوند؟
- آیا Anti-Passback برای ورودی‌ها یا مناطق خاص لازم است؟

## 4. دانش‌آموزان
- تعریف کلاس مجاز برای هر دانش‌آموز چگونه است؟
- آیا دسترسی دانش‌آموزان بر اساس برنامه هفتگی تغییر می‌کند؟
- در زنگ تفریح دانش‌آموزان به چه نواحی مجاز هستند؟
- کدام طبقات یا فضاها برای دانش‌آموزان ممنوع است؟
- آیا ورود به آزمایشگاه‌ها فقط در زمان کلاس مجاز است؟
- آیا حضور دانش‌آموز در کلاس باید ثبت شود یا فقط ورود به ساختمان کافی است؟

## 5. والدین و گزارش‌ها
- آیا والدین باید پنل مستقل داشته باشند؟
- آیا والدین فقط گزارش ورود و خروج روزانه را می‌بینند؟
- آیا اعلان لحظه‌ای برای ورود/خروج دانش‌آموز لازم است؟
- آیا این قابلیت باید از طریق API به سامانه موجود مدرسه متصل شود؟

## 6. مهمانان و مراجعان
- فرآیند ورود والدین، مهمانان و مراجعان چگونه است؟
- آیا تأیید میزبان قبل از ورود لازم است؟
- آیا QR، کارت موقت، تشخیص چهره یا ثبت دستی برای مهمانان مورد نظر است؟
- آیا خودرو مهمان باید به مجوز مهمان متصل شود؟

## 7. پارکینگ
- تعداد ورودی و خروجی پارکینگ چقدر است؟
- ظرفیت پارکینگ چقدر است؟
- آیا پارکینگ معلمان، مهمانان و سرویس مدرسه جداست؟
- اولویت روش کنترل خودرو چیست: ANPR، RFID یا UHF؟
- آیا Barrier Gate در همه ورودی‌ها لازم است؟

## 8. CCTV و تشخیص چهره
- آیا CCTV فعلی وجود دارد؟
- آیا دوربین‌های فعلی قابل استفاده هستند؟
- آیا طراحی کامل CCTV در Scope این پروپوزال است؟
- کدام راهروها، طبقات و نقاط حساس نیاز به تشخیص چهره دارند؟
- آیا تشخیص Tailgating یا عبور چند نفر با یک مجوز لازم است؟

## 9. تجهیزات بازرسی و کنترل اقلام
- آیا فلزیاب در ورودی‌ها لازم است؟
- آیا X-Ray برای کیف یا بسته‌ها لازم است؟
- آیا کنترل موبایل یا اقلام ممنوعه مورد انتظار است؟
- این تجهیزات در Scope اصلی باشند یا Option توسعه آینده؟

## 10. زیرساخت IT
- وضعیت شبکه داخلی چیست؟
- آیا PoE در محل موجود است؟
- آیا سرور اختصاصی موجود است؟
- آیا سیستم باید On-Premise نصب شود یا Cloud/Hybrid؟
- آیا Active Directory یا سامانه هویتی موجود است؟
- آیا سامانه مدرسه API یا بانک اطلاعاتی قابل اتصال دارد؟

## 11. خروجی نهایی
- آیا پروپوزال نهایی شامل BOQ دقیق باشد؟
- آیا قیمت‌گذاری لازم است؟
- آیا ارائه مدیریتی PowerPoint نیز نیاز است؟
- آیا پروپوزال باید شامل سه سناریوی اقتصادی/استاندارد/پیشرفته باشد؟
