# فرضیات پروژه

این سند شامل فرضیات فعلی پروژه است. فرضیات باید در جلسات بعدی با کارفرما بررسی و در صورت تأیید به تصمیم قطعی تبدیل شوند.

## 1. فرضیات عمومی
- پروژه مربوط به یک مدرسه یا مجموعه آموزشی بزرگ است.
- راهکار پیشنهادی توسط HLWL ارائه می‌شود.
- راهکارها و تجهیزات پیشنهادی بر پایه برند Syncolony خواهند بود.
- نسخه اولیه اسناد فارسی است و نسخه نهایی انگلیسی پس از تأیید محتوای فارسی تولید می‌شود.
- Markdown مرجع اصلی مستندات است و Word/PDF خروجی نهایی خواهند بود.

## 2. فرضیات محدوده پروژه
- کنترل دسترسی برای همه درب‌ها و فضاهای اصلی در نظر گرفته می‌شود.
- همه کلاس‌ها، آزمایشگاه‌ها، اتاق‌های اداری، فضاهای ورزشی، فضاهای حساس و ورودی‌ها در Scope بررسی قرار دارند.
- فاز اول پروپوزال شامل قیمت‌گذاری قطعی نیست، مگر اینکه کارفرما اطلاعات کافی و BoQ نهایی ارائه کند.
- تعداد نهایی تجهیزات پس از تحلیل پلان و بازدید یا تأیید کارفرما مشخص می‌شود.

## 3. فرضیات کاربری
- دانش‌آموزان بر اساس کلاس، پایه، برنامه زمانی و بخش مدرسه کنترل می‌شوند.
- پرسنل، معلمان و مدیران دسترسی متفاوت دارند.
- والدین و مهمانان دسترسی موقت و محدود خواهند داشت.
- پیمانکاران و سرویس مدارس به عنوان گروه‌های مستقل قابل تعریف هستند.

## 4. فرضیات فنی
- راهکار نرم‌افزاری امکان تعریف نامحدود گروه، نقش و سطح دسترسی را دارد.
- سیستم قابلیت یکپارچه‌سازی با سامانه‌های مدرسه از طریق API یا Web Service را خواهد داشت، در صورت فراهم بودن زیرساخت سمت کارفرما.
- امکان استفاده از تشخیص چهره، کارت RFID و تگ UHF در طراحی لحاظ می‌شود.
- اتصال به اعلام حریق، CCTV، فلزیاب و X-Ray فعلاً به عنوان قابلیت قابل بررسی یا Option در نظر گرفته می‌شود.

## 5. فرضیات مبتنی بر پلان
- پلان‌های Site Setting، Ground Floor و First Floor مبنای اولیه تحلیل هستند.
- پلان‌ها برای تخمین اولیه نقاط نصب تجهیزات کافی هستند، اما برای BoQ نهایی باید توسط کارفرما تأیید شوند.
- فایل AutoCAD در صورت دریافت، دقت جانمایی و شمارش تجهیزات را افزایش می‌دهد.

## 6. فرضیات تجاری
- پیشنهاد اولیه باید راهکار پیشنهادی اصلی را واضح و ساده ارائه کند.
- گزینه‌های جایگزین و توسعه‌ای باید جداگانه به عنوان Option معرفی شوند.
- در فاز نهایی امکان ارائه سه سطح راهکار وجود دارد:
  - اقتصادی
  - استاندارد
  - کامل / پیشرفته
