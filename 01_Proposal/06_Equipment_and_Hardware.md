# تجهیزات و سخت‌افزار پیشنهادی

## هدف سند

این سند چارچوب اولیه تجهیزات و سخت‌افزارهای پیشنهادی برای پروژه سیستم یکپارچه کنترل تردد، کنترل دسترسی، مدیریت امنیت و پارکینگ مدرسه را تشریح می‌کند.

هدف این سند، ارائه یک نگاه ساختاریافته به دسته‌بندی تجهیزات، کاربرد هر تجهیز، محل‌های احتمالی استفاده و وضعیت قطعی یا اختیاری بودن آن‌هاست. این سند در این مرحله جایگزین BOQ نهایی نیست و تعداد دقیق تجهیزات پس از تحلیل پلان AutoCAD، بررسی میدانی، تأیید مسیرهای تردد و نهایی شدن Scope پروژه تعیین خواهد شد.

> وضعیت سند: نسخه اولیه مفهومی  
> مبنای تهیه: نیازمندی‌های اولیه، سناریوهای عملیاتی، تحلیل پلان‌ها و مذاکرات پروژه  
> برند پیشنهادی تجهیزات: Syncolony، مگر در مواردی که به‌صورت Option یا Integration ذکر شود

---

## 1. رویکرد پیشنهادی در معرفی تجهیزات

با توجه به تنوع تجهیزات کنترل تردد و امنیت فیزیکی، پیشنهاد می‌شود در نسخه اولیه پروپوزال از ورود بیش از حد به مدل‌های متعدد تجهیزات خودداری شود و راهکار به شکل ساده، قابل فهم و مبتنی بر سناریوهای مدرسه ارائه گردد.

### اصول پیشنهادی

- ابتدا نیاز و سناریو توضیح داده شود.
- سپس نوع تجهیز موردنیاز معرفی شود.
- مدل دقیق تجهیز فقط در صورت نیاز و در بخش فنی یا ضمیمه ذکر شود.
- تجهیزات اصلی از تجهیزات اختیاری جدا شوند.
- تجهیزات بازرسی مانند فلزیاب، X-Ray و کنترل موبایل فعلاً به عنوان Option مطرح شوند، مگر اینکه کارفرما آن‌ها را صراحتاً در Scope اصلی قرار دهد.
- BOQ نهایی پس از تحلیل پلان و تأیید کارفرما تهیه شود.

## 3. تجهیزات کنترل ورود نفرات

### کاربرد

این تجهیزات برای کنترل ورود و خروج افراد از نقاط اصلی مانند ورودی مدرسه، ورودی بخش‌ها، خروجی‌های دانش‌آموزان، مسیرهای پررفت‌وآمد و نقاط حساس استفاده می‌شوند.

### تجهیزات قابل استفاده

| تجهیز | کاربرد | توضیح |
|---|---|---|
| Speed Gate | کنترل سریع و حرفه‌ای ورود نفرات | مناسب ورودی‌های اصلی و پررفت‌وآمد |
| Swing Gate | کنترل ورود با ظاهر مناسب و عبور راحت‌تر | مناسب ورودی‌های مدرسه، مهمانان یا مسیرهای خاص |
| Flap Gate | کنترل عبور سریع در محیط داخلی | مناسب ورودی‌های کنترل‌شده |
| Tripod Turnstile | کنترل عبور تک‌نفره اقتصادی‌تر | مناسب سناریوهای ساده‌تر یا مسیرهای با امنیت متوسط |
| Access Control Terminal | احراز هویت و صدور فرمان دسترسی | قابل استفاده کنار گیت یا درب |
| RFID Reader | کنترل با کارت | روش ساده و قابل استفاده برای گروه‌های مختلف |
| Face Recognition Terminal | احراز هویت بدون تماس | مناسب ورودی‌ها و فضاهای حساس |

### نقاط احتمالی استفاده

- ورودی اصلی مدرسه
- ورودی بخش پسران
- ورودی بخش دختران
- KG Student Exit
- Boys Exit
- Girls Exit
- Reception / Guard Room
- مسیرهای کنترل‌شده به Administration
- مسیرهای خاص منتهی به فضاهای حساس

### ابهامات

- تعداد گیت‌ها باید بر اساس ظرفیت تردد و عرض ورودی مشخص شود.
- نوع گیت باید بر اساس سطح امنیت، ظاهر، بودجه و میزان تردد انتخاب شود.
- تشخیص Tailgating در ورودی‌ها باید توسط کارفرما تأیید شود.

## 4. تجهیزات کنترل درب

### کاربرد

این تجهیزات برای کنترل دسترسی به کلاس‌ها، آزمایشگاه‌ها، اتاق‌های اداری، اتاق‌های حساس و سایر فضاهای داخلی استفاده می‌شوند.

### پکیج پایه پیشنهادی برای هر درب کنترل‌شونده

| تجهیز | کاربرد |
|---|---|
| Door Controller | مدیریت تصمیم دسترسی و کنترل قفل |
| Reader یا Face Terminal | شناسایی کاربر |
| Electric Lock یا Magnetic Lock | قفل کردن و آزادسازی درب |
| Door Sensor | تشخیص وضعیت باز یا بسته بودن درب |
| Exit Button | خروج کنترل‌شده از داخل |
| Emergency Break Glass | آزادسازی اضطراری در شرایط خاص |
| Power Supply | تأمین برق قفل و کنترلر |
| Network / PoE | اتصال به نرم‌افزار مرکزی، در صورت نیاز |

### انواع فضاهای قابل تجهیز

| فضا | سطح کنترل پیشنهادی |
|---|---|
| کلاس‌ها | کنترل دانش‌آموز و معلم |
| آزمایشگاه‌ها | کنترل سخت‌گیرانه و زمان‌بندی‌شده |
| Computer Lab | کنترل دسترسی و مانیتورینگ |
| IT Room | کنترل بسیار محدود |
| Electric Room | کنترل بسیار محدود |
| Administration | کنترل پرسنل |
| Manager Office | کنترل مدیریتی |
| Clinic | کنترل محدود |
| Library | کنترل زمان‌بندی‌شده |
| Gym | کنترل زمان‌بندی‌شده |
| Store / Lab Store / Control Store | کنترل محدود |
| Meeting Hall | کنترل رویدادی یا زمان‌بندی‌شده |

### ابهامات

- نوع واقعی درب‌ها مشخص نیست.
- جهت بازشو و امکان نصب قفل باید بررسی شود.
- برخی درب‌ها ممکن است به کنترل کامل نیاز نداشته باشند.
- برای درب‌های اضطراری باید سیاست Fail-Safe / Fail-Secure مشخص شود.

## 5. تجهیزات احراز هویت افراد

### روش‌های اصلی

| روش | کاربرد | وضعیت پیشنهادی |
|---|---|---|
| تشخیص چهره | ورود سریع، بدون تماس و مناسب مدرسه | اصلی |
| کارت RFID | روش ساده، ارزان‌تر و پشتیبان | اصلی |
| PIN | روش مکمل یا اضطراری | Option |
| QR Code | مناسب مهمانان و مجوز موقت | Option |
| اثر انگشت | قابل ارائه در صورت نیاز | Option |
| موبایل / Mobile Credential | قابل بررسی در فاز توسعه | Option |

### رویکرد پیشنهادی

برای ساده‌سازی Proposal، پیشنهاد می‌شود در نسخه اول، احراز هویت افراد بر پایه ترکیب زیر معرفی شود:

- دانش‌آموزان: Face Recognition یا RFID، بسته به سیاست مدرسه
- پرسنل و معلمان: Face Recognition + RFID
- مدیران و فضاهای حساس: Face Recognition + کارت یا روش ترکیبی
- مهمانان: کارت موقت، QR یا ثبت در Visitor Management
- پیمانکاران: مجوز موقت زمان‌دار

### ابهامات

- سیاست استفاده از چهره برای دانش‌آموزان باید از نظر حقوقی و اجرایی بررسی شود.
- آیا کارت برای همه دانش‌آموزان صادر می‌شود؟
- آیا اثر انگشت در محیط مدرسه قابل قبول است؟
- آیا والدین یا مهمانان از QR استفاده می‌کنند؟

این تجهیزات برای کنترل حضور افراد در راهروها، طبقات و نقاطی استفاده می‌شوند که دسترسی غیرمجاز به آن‌ها باید تشخیص داده شود.

- دوربین در ورودی آزمایشگاه‌ها
- دوربین در ورودی بخش‌های پسران و دختران
- دوربین در Drop Area و خروجی‌های دانش‌آموزان
- دوربین در پارکینگ و ورودی خودروها

### نقاط احتمالی نصب

- Shaded Corridors
- مسیرهای بین بخش‌ها
- راهروهای منتهی به آزمایشگاه‌ها
- ورودی Administration
- ورودی IT Room و فضاهای حساس
- KG Zone
- Boys Section
- Girls Section
- Drop Area
- Teacher Parking Area

### ابهامات

- وضعیت CCTV موجود مشخص نیست.

- تشخیص حضور در ناحیه غیرمجاز نیازمند طراحی دقیق نرم‌افزار و سیاست‌های Area است.
- هر دوربین IP به شبکه و برق نیاز دارد.

## 7. تجهیزات پارکینگ

### کاربرد

تجهیزات پارکینگ برای کنترل ورود و خروج خودروهای مجاز، پرسنل، مهمانان، سرویس مدرسه و خودروهای خدماتی استفاده می‌شوند.

### تجهیزات پیشنهادی

| تجهیز | کاربرد |
|---|---|
| Barrier Gate | کنترل فیزیکی ورود و خروج خودرو |
| ANPR Camera | شناسایی پلاک خودرو |
| UHF Reader | شناسایی تگ خودرو از فاصله بیشتر |
| UHF Tag | مجوز خودرو برای پرسنل یا سرویس |
| RFID Reader | روش مکمل برای کنترل خودرو |
| Loop Detector / Vehicle Sensor | تشخیص حضور خودرو |
| Parking Controller | مدیریت فرمان Barrier و ارتباط با نرم‌افزار |
| Traffic Light | هدایت خودروها در صورت نیاز |
| CCTV Camera | ثبت تصویر خودرو و مسیر پارکینگ |

### سناریوهای قابل پوشش

- ورود پرسنل با پلاک یا تگ UHF
- ورود مهمان با مجوز موقت
- ورود سرویس مدرسه
- ورود خودروهای خدماتی
- ثبت رویدادهای ورود و خروج
- گزارش‌گیری از خودروها
- اتصال خودرو به مالک یا مجوز مهمان

### ابهامات

- تعداد ورودی و خروجی پارکینگ مشخص نیست.
- ظرفیت پارکینگ مشخص نیست.
- مسیر دقیق خودروهای پرسنل و مهمانان باید از روی پلان و بازدید مشخص شود.
- مشخص نیست خودروهای والدین اجازه ورود دارند یا فقط از Drop Area استفاده می‌کنند.

### کاربرد

برای ثبت، کنترل و مدیریت ورود والدین، مهمانان، مراجعان، پیمانکاران و خودروهای مرتبط با آن‌ها استفاده می‌شود.

### تجهیزات و ابزارهای احتمالی

| تجهیز / ابزار | کاربرد |
|---|---|
| Workstation در Guard Room یا Reception | ثبت اطلاعات مهمان |
| کارت‌خوان یا QR Reader | صدور و کنترل مجوز موقت |
| وب‌کم یا دوربین ثبت تصویر | ثبت تصویر مراجعه‌کننده |
| پرینتر کارت یا Badge، در صورت نیاز | چاپ کارت مهمان |

| Reader یا Gate در مسیر مهمان | کنترل عبور مهمان به ناحیه مجاز |

### ابهامات

- آیا کارت مهمان چاپ می‌شود یا فقط QR/کارت موقت کافی است؟
- آیا تأیید میزبان لازم است؟
- آیا خودرو مهمان باید به درخواست مراجعه متصل شود؟
- آیا خروج مهمان باید ثبت و تأیید شود؟

## 10. تجهیزات بازرسی و کنترل اقلام

### وضعیت پیشنهادی

این بخش در حال حاضر باید به عنوان Option یا قابلیت توسعه آینده در نظر گرفته شود، مگر اینکه کارفرما آن را صراحتاً به محدوده اصلی پروژه اضافه کند.

### تجهیزات قابل ارائه به عنوان Option

| تجهیز | کاربرد |
|---|---|
| Walk Through Metal Detector | کنترل عبور نفرات و تشخیص فلزات |
| Handheld Metal Detector | بازرسی دستی |
| X-Ray Inspection System | کنترل کیف، بسته و اقلام همراه |
| Mobile Phone Detection / Control | کنترل موبایل یا اقلام ممنوعه، در صورت وجود راهکار مناسب |

### ابهامات

- آیا کنترل موبایل واقعاً نیاز کارفرماست؟
- آیا تجهیزات بازرسی در ورودی مدرسه لازم است؟
- آیا X-Ray برای مدارس از نظر عملیاتی و فرهنگی قابل قبول است؟
- آیا این تجهیزات باید در پروپوزال اصلی یا ضمیمه Options بیایند؟

## 11. تجهیزات زیرساختی

### شبکه

#### تجهیزات وابسته به شبکه

- Face Recognition Terminal
- IP Camera
- ANPR Camera
- Access Controller
- Gate Controller
- Parking Controller
- Workstation
- Server
- Switch
- NVR یا VMS، در صورت وجود CCTV

#### نیازهای شبکه

- نود شبکه در محل نصب
- کابل‌کشی مناسب
- PoE، در صورت نیاز
- رک شبکه
- VLAN امنیتی
- پهنای باند کافی
- دسترسی امن برای پشتیبانی
- مانیتورینگ وضعیت اتصال تجهیزات

### برق

تجهیزات زیر نیاز به برق پایدار دارند:

- گیت‌ها
- کنترلرها
- قفل‌ها
- ترمینال‌ها
- دوربین‌ها
- Barrier Gate
- سرور
- سوئیچ‌ها
- Workstationها

### UPS

برای تجهیزات حیاتی، UPS باید بررسی شود:

- سرور
- سوئیچ‌های اصلی
- گیت‌های ورودی
- کنترلرهای حیاتی
- تجهیزات پارکینگ
- Guard Room / Reception

### ابهامات

- آیا شبکه فعلی مدرسه آماده است؟
- آیا PoE در محل موجود است؟
- آیا سرور اختصاصی موجود است؟
- آیا برق اضطراری برای گیت‌ها و درب‌های حساس لازم است؟
- آیا زیرساخت شبکه و برق در محدوده قیمت‌گذاری HLWL خواهد بود؟

## 14. سطح‌بندی پیشنهادی راهکار

در فاز نهایی و در صورت نیاز کارفرما، راهکار می‌تواند در سه سطح ارائه شود.

### سطح 1: اقتصادی

- کنترل ورودی‌های اصلی
- کنترل فضاهای حساس
- RFID / کارت به عنوان روش اصلی
- پارکینگ با RFID یا UHF
- گزارش‌گیری پایه

### سطح 2: استاندارد

- کنترل ورودی‌ها، کلاس‌ها و فضاهای حساس
- Face Recognition + RFID
- Visitor Management
- Parking با ANPR / UHF
- CCTV در نقاط اصلی
- گزارش‌گیری و هشدار پایه

### سطح 3: کامل / پیشرفته

- کنترل کامل همه درب‌ها و کلاس‌ها
- Face Recognition گسترده در راهروها
- Parent Portal / API Integration
- تشخیص حضور در نواحی غیرمجاز
- Tailgating Detection
- تجهیزات بازرسی، در صورت نیاز
- مانیتورینگ پیشرفته و گزارش‌های مدیریتی
