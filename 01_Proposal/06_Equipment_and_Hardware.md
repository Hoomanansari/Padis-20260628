# تجهیزات و سخت‌افزار پیشنهادی

## 1. هدف سند
این سند چارچوب اولیه تجهیزات مورد نیاز پروژه را معرفی می‌کند. تعداد نهایی تجهیزات پس از تحلیل دقیق پلان، بازدید میدانی و تأیید کارفرما مشخص خواهد شد.

## 2. دسته‌بندی تجهیزات

### 2.1 تجهیزات کنترل تردد نفرات
- Speed Gate
- Swing Gate
- Flap Gate
- Tripod Turnstile
- Access Control Terminal
- Face Recognition Terminal
- RFID Reader
- Door Controller
- Electric Lock / Magnetic Lock
- Exit Button
- Emergency Break Glass
- Door Sensor

### 2.2 تجهیزات تشخیص چهره و احراز هویت
- Face Recognition Access Control Terminal
- دوربین یا ترمینال تشخیص چهره برای ورودی‌ها و نقاط حساس
- تجهیزات ثبت چهره و کارت در صورت نیاز

### 2.3 تجهیزات کنترل درب
برای هر درب کنترل‌شونده، بسته به نوع درب، تجهیزات زیر قابل بررسی است:

- Door Controller
- Reader یا Face Terminal
- Electric Lock یا Magnetic Lock
- Door Contact
- Exit Button
- Power Supply
- Network/PoE در صورت نیاز

### 2.4 تجهیزات پارکینگ
- Barrier Gate
- ANPR Camera
- RFID Reader
- UHF Reader
- UHF Tag
- Loop Detector یا Vehicle Sensor
- Controller
- Traffic Light در صورت نیاز

### 2.5 تجهیزات مانیتورینگ و امنیت
- CCTV Camera
- IP Camera برای ثبت تصویر تردد
- دوربین تشخیص چهره در راهروها
- مانیتورینگ مرکزی
- هشدار نرم‌افزاری

### 2.6 تجهیزات بازرسی اختیاری
این تجهیزات فعلاً به عنوان Option یا قابلیت توسعه آینده قابل بررسی هستند:

- Walk Through Metal Detector
- Handheld Metal Detector
- X-Ray Inspection System
- تجهیزات کنترل موبایل یا اقلام ممنوعه

## 4. رویکرد پیشنهادی برای انتخاب تجهیزات
برای جلوگیری از پیچیدگی غیرضروری در ارائه اولیه، پیشنهاد می‌شود:

1. یک راهکار اصلی و ساده برای کارفرما ارائه شود.
2. گزینه‌های پیشرفته‌تر به‌صورت Option معرفی شوند.
3. تجهیزات اختیاری مانند X-Ray، فلزیاب، کنترل موبایل و سناریوهای پیشرفته در بخش توسعه آینده یا Alternative Options بیایند.
4. تعداد نهایی تجهیزات پس از تحلیل پلان و تأیید کارفرما تعیین شود.

## 5. ابهامات تجهیزات
- تعداد نهایی درب‌ها و مسیرهای کنترل‌شونده مشخص نیست.
- نوع درب‌ها و امکان نصب قفل الکترونیکی باید بررسی شود.
- وضعیت شبکه، برق و PoE مشخص نیست.
- وضعیت CCTV موجود مشخص نیست.
- تجهیزات بازرسی باید به عنوان Scope اصلی یا Option تعیین شوند.
- اولویت استفاده از Face، RFID یا ترکیبی برای هر گروه کاربری باید تعیین شود.
